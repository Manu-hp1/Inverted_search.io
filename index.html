Inverted Search
Engine

A C-based keyword search system that retrieves files instantly using an inverted index, eliminating repeated file scanning.

Source Code Watch Demo

Core Concept

Inverted Index

Maps each word to files where it appears.

Fast Retrieval

Search without scanning every file.

Efficient Storage

Stores word frequency and file references.

Inverted Index Representation

data
→
file1(3)
file2(1)

Each word points to a list of files along with occurrence count.

Search Flow

Input Keyword
↓
Search in Index
↓
Retrieve File List
↓
Display Results

Key Features

  • Fast keyword search using indexing
  • Stores word occurrences per file
  • Efficient file parsing
  • Dynamic memory allocation
  • Modular design for indexing & searching
