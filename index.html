Inverted Search
Engine

A C-based keyword search system that retrieves files instantly using an inverted index instead of repeatedly scanning files.

Source Code Watch Demo

Core Concept

Inverted Index

Maps every word to files where it appears.

Fast Retrieval

Eliminates repeated full-file scanning.

Efficient Search

Stores word frequency and file references.

Inverted Index Representation

data
→
file1(3)
file2(1)

Each word points to the files where it exists along with occurrence count.

Search Flow

Input Keyword
↓
Search in Index
↓
Retrieve Matching Files
↓
Display Results

Key Features

  • Fast keyword-based retrieval
  • Efficient indexing using hashing
  • Dynamic memory allocation
  • File parsing and string processing
  • Modular code structure

Technology Stack

C Programming Linked Lists Hashing File Handling Dynamic Memory String Processing
